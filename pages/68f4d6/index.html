基于docker的mysql5.7主从复制测试

show master status;

show slave status;

flush privileges;

grant replication slave, replication client on *.* to 'repl'@'%' identified by 'password';

change master to master_host='192.168.60.10',master_user='repl',master_password='password',master_port=3306,master_log_file='mysql-bin.000001',master_log_pos=449;

start slave;

show global variables like "%read_only%";

flush tables with read lock;
set global read_only=1;

unlock tables;
set global read_only=0;

show processlist;

select * from test.stu_name;

配置mysql5.7容器中/etc/mysql/mysql.conf.d/mysqld.cnf

主要配置项：

server-id=1

log-bin=mysql-bin //主库配置

relay_log=xxx

开启mysql5.7容器：

docker run -p 3306:3306 --name s1 -e MYSQL_ROOT_PASSWORD=password -d mysql:5.7

注意配置更新重启mysql容器。
